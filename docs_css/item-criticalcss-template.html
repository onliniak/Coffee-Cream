Here, I create secend important part of this site - typography.

I use 3 Helvetica's fonts → more info http://www.identifont.com/differences?first=Nimbus+Sans&second=Helvetica and http://www.identifont.com/differences?first=Arial&second=Helvetica&q=Go and "automatic/mathematic" font size.

14px + (¼*%*x) or 14px + (0.25*0.01*x)
Where x =  height of article/content area.

For example on laptops (1366x768px) we use ~ 17.3px I am using 150% since I noticed that the "automatic" version causes problems with larger fonts like 30 / 40px And colors - dark grey for text and links.
